## Sobre

Um client expresso para operar um Bucket na AWS S3, com uma interface minimalista e intuitiva.

## Tecnologias

- Java 17
- Maven
- JavaFX
- SceneBuilder
- AWS SDK v2 - Java

## Features

- [x] Login
  - [x] Salvar sessão
- [x] Gerenciamento de objetos
  - [x] Listagem
  - [x] Criação
  - [ ] Deleção

## Como rodar 
> Aplicação nativa em desenvolvimento. Por enquanto, utilize Maven para rodar.

1. Clone o projeto

    ````bash
    git clone https://github.com/filipemartinsdev/j-bucket
    cd j-bucket
    ````

3. Inicie a aplicação

    ````bash
    mvn javafx:run
    ````
